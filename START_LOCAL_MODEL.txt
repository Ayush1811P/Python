🚀 INSTALLATION STEPS (5 Minutes)
═══════════════════════════════════════════════════════════════════════════════════

STEP 1: Download & Install Ollama
──────────────────────────────────────────────────────────────────────────────────
Visit: https://ollama.ai
Select your OS (Windows/Mac/Linux)
Follow the installer

Verify installation:
  ollama --version


STEP 2: Download TinyLlama Model
──────────────────────────────────────────────────────────────────────────────────
In terminal/command prompt:
  
  ollama pull tinyllama

This downloads ~600MB. Wait for "success" message.

Verify:
  ollama list
  (You should see tinyllama in the list)


STEP 3: Start Ollama Server
──────────────────────────────────────────────────────────────────────────────────
In a SEPARATE terminal (keep it open):
  
  ollama serve

You should see:
  "Listening on [::1]:11434"

⚠️  IMPORTANT: Keep this terminal open while using the agent!


STEP 4: Install Python Packages
──────────────────────────────────────────────────────────────────────────────────
In NEW terminal (not the Ollama one):
  
  pip install -r requirements_agent.txt

Should see:
  "Successfully installed python-dotenv requests"


STEP 5: Setup Configuration
──────────────────────────────────────────────────────────────────────────────────
Copy template file:
  
  cp .env.example .env

Open .env and verify:
  OLLAMA_BASE_URL=http://localhost:11434
  LOCAL_MODEL=tinyllama


STEP 6: Run the Agent
──────────────────────────────────────────────────────────────────────────────────
In NEW terminal (Ollama still running in separate terminal):
  
  python ai_agent.py

You should see:
  ✓ Connected to Ollama
  ✓ Model 'tinyllama' is available
  ✓ Agent initialized successfully!

🚨 COMMON ISSUES & FIXES
═══════════════════════════════════════════════════════════════════════════════════

Issue: "Cannot connect to Ollama"
Fix:   1. Start Ollama: ollama serve
       2. Keep terminal open
       3. Check: curl http://localhost:11434/api/tags

Issue: "Model 'tinyllama' not found"
Fix:   1. Pull model: ollama pull tinyllama
       2. Wait for "success"
       3. Verify: ollama list

Issue: Agent responds slowly
Fix:   1. First response takes time (loading model)
       2. Subsequent responses faster (~2-5 sec)
       3. Can use faster model: ollama pull neural-chat

Issue: "Timeout" or hanging
Fix:   1. Check system RAM (need 4GB+)
       2. Close other applications
       3. Restart Ollama server


💡 PERFORMANCE TIPS
═══════════════════════════════════════════════════════════════════════════════════

Speed Up:
  • Use smaller model: ollama pull phi
  • Use quantized models (already using Q4_K_M)
  • Ask simpler questions
  • Close other apps

Improve Quality:
  • Use larger model: ollama pull mistral
  • Ask more specific questions
  • Provide more context

Reduce Memory:
  • Use smaller model: ollama pull orca-mini
  • Monitor system resources
  • Don't run other heavy apps
