STEP 1: VERIFY SETUP
────────────────────────────────────────────────────────────────────────────────
Run the verification script:
    python test_agent.py

This checks:
  ✅ Python version
  ✅ Required packages installed
  ✅ File structure
  ✅ .env configuration
  ✅ Agent code structure


STEP 2: CONFIGURE CREDENTIALS
────────────────────────────────────────────────────────────────────────────────
A) Create .env file:
    cp .env.example .env

B) Add OpenAI API Key:
    1. Visit: https://platform.openai.com/api-keys
    2. Create a new API key
    3. Copy it
    4. Open .env and replace: OPENAI_API_KEY=sk-...

C) (OPTIONAL) Add Gmail for Email Sending:
    1. Enable 2FA: https://support.google.com/accounts/answer/185839
    2. Generate app password: https://myaccount.google.com/apppasswords
    3. Select "Mail" and "Windows Computer"
    4. Copy the 16-character password
    5. Add to .env: SENDER_PASSWORD=your_password
    6. Add .env: SENDER_EMAIL=your_email@gmail.com


STEP 3: RUN THE AGENT
────────────────────────────────────────────────────────────────────────────────
Start interactive mode:
    python ai_agent.py

Then try these commands:
    • "What can you do?"
    • "Create a task to review the budget"
    • "List my pending tasks"
    • "Draft an email to john@company.com"

QUICK TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────────
Problem: "ModuleNotFoundError: No module named openai"
Fix:     pip install openai python-dotenv

Problem: "OPENAI_API_KEY not set"
Fix:     Create .env file with OPENAI_API_KEY=sk-...

Problem: "Email sending fails"
Fix:     Check SENDER_EMAIL/SENDER_PASSWORD in .env

Problem: "Agent doesn't respond"
Fix:     Check internet connection & API key validity

Problem: "No module named 'dotenv'"
Fix:     pip install python-dotenv

See FAQ_AND_TROUBLESHOOTING.py for more help!
