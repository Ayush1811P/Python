## 🚀 Getting Started (3 Steps)

### Step 1: Install Dependencies
```bash
pip install -r requirements_agent.txt
```

### Step 2: Setup Credentials
```bash
# Copy template
cp .env.example .env

# Edit .env and add your:
# - OPENAI_API_KEY (from https://platform.openai.com/api-keys)
# - SENDER_EMAIL (Gmail address)
# - SENDER_PASSWORD (Gmail app password - see README)
```

### Step 3: Run
```bash
# Interactive agent
python ai_agent.py

# Run demos
python demo_agent.py

# Verify setup
python quick_start.py

# See usage examples
python usage_examples.py
```

## 🔧 Architecture

```
User Input (natural language)
    ↓
OpenAI GPT-4 (analyzes intent)
    ↓
Selects appropriate tools
    ↓
EmailManager / TaskManager execute
    ↓
Returns results to model
    ↓
Natural language response
    ↓
User sees result
```

## 🛠️ Customization

### Use Different AI Model
Edit `ai_agent.py`, line ~135:
```python
self.model = "gpt-4"  # Change to gpt-4-turbo or gpt-3.5-turbo
```

### Add Custom Tools
1. Create new tool in EmailManager or TaskManager
2. Add tool definition to `self.tools[]` in AIAgent.__init__()
3. Add handler in AIAgent.process_tool_call()

### Store Data Persistently
Replace TaskManager's in-memory dict with:
- SQLite database
- PostgreSQL
- MongoDB
- File storage (JSON/CSV)

## 🐛 Troubleshooting

| Issue | Solution |
|-------|----------|
| "Invalid API key" | Check OPENAI_API_KEY in .env |
| "Email send failed" | Verify Gmail credentials + generate app password |
| "Module not found" | Run `pip install -r requirements_agent.txt` |
| "No response from agent" | Ensure internet connection + OpenAI API access |
