DATA FLOW EXAMPLE: "Create task and email the client"
═══════════════════════════════════════════════════════════════════════════════

    1. User: "Create task to follow up with John and email him about it"
                │
    2. AIAgent ──► OpenAI GPT-4
                │ (Analyzes: multiple tools needed)
                │ (Decides: create_task + send_email)
                │
    3. GPT-4 ────► AIAgent.process_tool_call()
                │ Tool 1: create_task(
                │           task_id='follow_up_john',
                │           title='Follow up with John',
                │           description='Email about the proposal'
                │         )
                │ Tool 2: send_email(
                │           recipient='john@example.com',
                │           subject='Proposal Follow Up',
                │           body='Hi John, wanted to follow up...'
                │         )
                │
    4. Execution ─► TaskManager creates task
                 ► EmailManager sends email
                 ► Returns results to GPT-4
                │
    5. GPT-4 ────► Generates natural response
                │ "I've created the task and sent John an email"
                │
    6. Output ───► User sees confirmation


STATELESS DESIGN PATTERN:
═══════════════════════════════════════════════════════════════════════════════

    Each conversation is independent but maintains:
    • Conversation history (for context)
    • Task state (in-memory dictionary)
    • Loaded environment variables


EXTENSION POINTS:
═══════════════════════════════════════════════════════════════════════════════

    To add new capabilities:
    
    1. Add Tool Definition in AIAgent.__init__()
       tools = [{
           "type": "function",
           "function": {
               "name": "new_tool",
               "description": "...",
               "parameters": {...}
           }
       }]
    
    2. Implement in Manager Class
       class NewManager:
           def new_tool(self, param1, param2):
               return {"status": "success", ...}
    
    3. Add Handler in process_tool_call()
       elif tool_name == "new_tool":
           return self.new_manager.new_tool(**tool_input)
