🚀 QUICK START (6 STEPS)
═══════════════════════════════════════════════════════════════════════════════════

1️⃣  Install Ollama
    Visit: https://ollama.ai
    Download for your OS

2️⃣  Pull TinyLlama Model
    ollama pull tinyllama
    (Wait for ~600MB download)

3️⃣  Start Ollama Server (KEEP OPEN)
    ollama serve
    (Keep this terminal running!)

4️⃣  Install Python Packages
    pip install -r requirements_agent.txt

5️⃣  Setup Configuration
    cp .env.example .env
    (Edit if needed)

6️⃣  Run Agent
    python ai_agent.py
    (In a NEW terminal)

💡 TIPS & TRICKS
═══════════════════════════════════════════════════════════════════════════════════

Speed:
  • First response is slower (loading model)
  • Keep Ollama running to avoid reload
  • Use smaller models for faster responses

Quality:
  • Ask clear, specific questions
  • TinyLlama is good for basic tasks
  • Use Mistral for complex tasks

Memory:
  • Monitor system RAM usage
  • Close other applications if slow
  • Use smaller models if needed


❓ TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════════

Can't connect to Ollama?
  → Start Ollama: ollama serve

Model not found?
  → Pull model: ollama pull tinyllama

Too slow?
  → First response loads model (~2-5 sec)
  → Try smaller model: ollama pull phi

Out of memory?
  → Check system RAM
  → Close other applications
  → Use smaller model
