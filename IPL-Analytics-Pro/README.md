# IPL Analytics Pro Project

## Overview
End-to-end cricket analytics project using Python, MySQL, and Power BI.

## Features
- Data Cleaning Pipeline
- SQL Analysis
- Power BI Dashboard
- DAX Measures

## Setup
1. Add dataset in /data
2. Run python/pipeline.py
3. Import cleaned data into Power BI
4. Use dax_measures.txt

## Tech Stack
Python, Pandas, NumPy, MySQL, Power BI
